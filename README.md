# Web Design Project Repository

## Student Wellness and Mental Health Website

This website is designed to support **student wellness and mental health** by providing accessible, reliable resources and information.

The project includes:

- Resources students can access for mental health support  
- Information about different mental health issues and disorders  
- Stories and advice specifically from **Charlotte Latin students**

## Purpose

The goal of this website is to:

- Raise awareness about mental health  
- Provide helpful, student-friendly resources  
- Create a supportive space for students seeking information or guidance  

## Features

- Curated mental health resources  
- Educational content on mental health conditions  
- Personal stories and advice from students  

## Target Audience

- Charlotte Latin students  
- Students seeking mental health information and support  

## Technologies Used

- HTML  
- CSS  
- JavaScript
